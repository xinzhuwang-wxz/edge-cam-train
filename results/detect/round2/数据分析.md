## 4. 命门 bird：已适配观鸟器域（本轮重点，质量好）

- **域**：feeder-cam（meproject 真定拍 + feeder UK 花园种）+ 清晰照 = 50.8%，网图降到 21%（原 50%）。
- **尺度**：中位 7.4%（"鸟落镜头前"中等尺度），远景噪声从 7525 框砍到 1679。
- **多样性**：5 域混合（定拍/清晰照/网图/相机陷阱/自然特写）→ 泛化好。
- **教师伪标注**：iNat 段（1327 框，md_pseudo 3%）经 MD≥0.7 auto 筛，闭环已验证（但域降级）。

## 5. ⚠️ Known Limitations（诚实标注，不阻塞本轮但需知晓）

1. **squirrel / cat 是单域（相机陷阱）+ 尺度偏小**：squirrel 中位 **1.3%**、cat **3.1%**（相机陷阱远景尺度）。
   观鸟器实景里松鼠/猫上喂食台会**更大更近**——当前无 feeder 域的 squirrel/cat 数据。这是最弱环。
   *影响*：模型对"喂食器上的大松鼠/猫"泛化可能弱。*缓解*：后续补 feeder 域 squirrel/cat（Roboflow/自采）。
2. **person 单域（OIV7 网图）**：非 feeder 场景的人。但"检到有人"易、且院子里的人视觉相近，风险较低。
3. **负样本单域（相机陷阱空帧）**：同为户外空场景、合理；理想再补**空喂食器帧**（更贴部署背景）。
4. **CC-BY 逐图署名**：现用数据集级 default_author（满足门 + 有效归属）；iNat/OIV7 逐图作者**已有取用能力**
   （iNat API / OIV image-ids CSV），**发行前**串入成逐图清册（§4 随产物披露），训练不阻塞。
5. **bird-v2 含 Roboflow 烘焙增强**（8658=3762 源+增强），已 `max_per_class:{bird:4000}` 压制；效果上等于
   额外增强，无泄漏（Roboflow 增强只在 train split）。
