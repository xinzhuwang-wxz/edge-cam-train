# 数据源审查 · 螺旋 harness 第一圈 dogfood

> **用审查闸门驱动数据源选型**——不是"列有哪些可商用源"，而是每个源过三 lens：③守红线(许可) / ②合现实(域匹配观鸟器现场) / ①服务命门(覆盖首发地域的种+长尾)。这是螺旋 harness v0 的**第一个真实 dogfood**（2026-07-06）。
> 承接 ADR-0005(数据许可)、[[06-螺旋harness-v0]]（命门+roadmap）。**核心产出 = 首发地域决策**（下 §四）。

## 二、三 lens 审查（闸门判据）

**③ 守红线（许可）——审出几个反直觉硬伤：**
- **iNat 是 ToS 契约问题、非 license 问题**：ToS 明禁"任何 iNat 数据训商用 AI"，**连 CC0 图都被额外契约挡**，换 head/换数据洗不白。→ 只 R&D，商用前必换/澄清（ADR-0005 决策1 正确）。
- **Observation.org 图默认 NC**（别和 eBird EOD 的 CC-BY 混）→ 图/先验两条线都排除。
- **birds525 的 CC0 是上传者对网络抓图的自我声明**（无权授权）→ 只 R&D。
- **Wikimedia 主流 CC-BY-SA**（ShareAlike 训练权重传染有争议）→ 只取 PD+CC-BY 子集。
- **欧洲三源 GBIF 媒体 license 是自由文本、多数未填** → 必须逐张 `media.license` 客户端过滤（可用图数 < 观测数）。

**② 合现实（域匹配观鸟器现场）——核心痛点，域真实性排序：**
> 自采 feeder ≫ 喂食器/相机陷阱开源集（域贴但多 iNat 派生、许可污染，只做 holdout）＞ 公民科学随手拍（iNat/naturgucker/挪/瑞，鸟偏小有遮挡，中）＞ 观鸟人精品图（Macaulay/Flickr/Wikimedia，对焦锐鸟大，域**最远**）＞ 博物馆标本（排除）。
- **关键洞察**：**所有非自采源都有域 gap**；Macaulay 这类"好图"补的是**种类覆盖**，不是**域真实性**。域真实性只能靠**自采 feeder** + 少量喂食器数据集做评测。合成/3D 有域 gap，仅增强。

**① 服务命门（覆盖首发地域种+长尾）：**
- 欧洲：naturgucker+挪+瑞 ~1.4M CC-BY，充分。
- 北美：去 iNat 后**商用干净实拍薄**，只剩 Macaulay CC-BY(未量化)/Flickr/Wikimedia/自采。
- 长尾/稀有种：Wikimedia/Flickr CC-BY iconic 图凑"每种至少几张" + 层级回退（<50 张 → genus/family/cloud）。

## 三、审查结论：红线内可用清单

- **可商用图源（进权重）**：naturgucker · 挪 · arter（CC-BY）· **Macaulay CC-BY 子集(需量化)** · Flickr CC0/CC-BY(需自标) · Wikimedia PD+CC-BY(慎 SA) · **自采 feeder**。逐张 license 过滤是一等步骤。
- **仅 R&D（商用前替换/澄清）**：iNaturalist CC0/CC-BY 子集。
- **可商用先验源**：GBIF Aves CC0/CC-BY（含 eBird EOD）· **FeederWatch（北美喂食器专属）**。
- **排除**：Observation.org(NC) · birds525(仅R&D) · NABirds/CUB · 博物馆标本 · eBird 丰度 · NC teacher。

## 四、审查回改路线（闸门 → roadmap 变更）★ 本圈核心产出

**首发地域决策（数据说话，服务命门）：**
- **✅ 欧洲首发 —— 数据支撑充分**：~1.4M CC-BY 实拍 + 欧洲 GBIF 先验，够撑一个可发行的欧洲包。**定：v1 首发欧洲。**
- **⚠️ 欧洲+北美同发 —— 图侧不支持**：去 iNat 后北美商用干净实拍薄，当前证据不支持北美与欧洲同质量同发。
- **推荐节奏：欧洲首发视觉模型 → 北美 fast-follow，前置一个 gate**（三选一达标才上北美视觉包）：
  1. Macaulay CC-BY 子集量化够用，或
  2. 自采北美 feeder 数据到量，或
  3. iNat 商用许可书面澄清。
- **北美"区域先验层"可早发**（FeederWatch+eBird EOD 齐备），符合 OTA 换表不换模型架构（ADR-0003）——北美用户先有"地域候选清单"，视觉包 fast-follow。

**新增入 roadmap 的动作：**
- **Macaulay CC-BY opt-in 子集量化 spike**（北美唯一规模化商用干净图候选，占比未公开、须自查）。
- **Project FeederWatch 纳入先验的北美层**（比通用 GBIF occurrence 更贴喂食器）。
- **自采 feeder 采集**（域真实性唯一来源，欧洲/北美都需要）。

**纠偏（文档/memory）：**
- `01-数据集.md`：补 Observation.org 排除（NC）；补 Macaulay CC-BY 子集 + FeederWatch 两个新增点（§2/§7）。
- birds525 定位：显式标注 CC0 为**上传者自我声明、网络抓取来源、仅 R&D**，勿升级为商用干净源。

## 五、闸门自审（这圈 harness 跑得怎样）

- **审查判据升级生效**：从"有哪些可商用源"(did it work) → "哪些源真服务命门+合域+守红线"(does it serve the goal)，审出了 iNat ToS 契约、域 gap 本质、北美缺口三个非显然结论，并**直接改了 roadmap**（定首发欧洲、新增 3 个动作）——这正是螺旋 harness 要的"审查→回改路线"。
- **待 harness 精修**：域匹配(②)目前靠定性判断，缺一个量化的"域 gap 指标"（网络图 vs 现场图的分布距离）——可在数据诊断(R1.2)里补，让②lens 有数可依。
