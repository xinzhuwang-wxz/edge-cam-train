## 二、数量溯源（"为什么 1000+ 变少"）

不同定义在不同步骤逐级收窄，都对，量的是不同东西：

| 数 | 含义 | 来源 |
|---|---|---|
| 11,167 | 全球所有鸟种 | eBird/Clements registry（云端天花板） |
| ~961 | 欧洲曾记录（含迷鸟） | Avibase Europe checklist |
| ~544 | 欧洲**常规**出现 | BirdLife regularly-occurring |
| **1,211** | eBird 45 国 checklist 并集 ∩ registry = **类集 universe**（含迷鸟/外来） | `data/region/europe/europe_species.jsonl` |
| **761** | 1211 里有**欧洲观测媒体**的（occurrence 级 CC-BY） | 初版清单（**已废**，见许可） |
| **502** | 761 里**媒体级真 CC0/CC-BY**的（商用干净） | **本清单** |

- 层级：502 ⊆ 761 ⊆ 1211 ⊆ 11,167。
- 缺的 709 种（1211−502）= 无商用干净媒体 → **genus 折叠**（粒度按数据，ADR-0008 §10）；主要是外来种（鹦鹉/waxbill）、美洲迷鸟（新大陆莺）、罕见尾种。

## 四、许可（③红线核心 —— 本次审计最重要修正）

**发现**：GBIF **occurrence 级** license（初版按它筛）**≠ 图片级** license。逐条查媒体级：
occurrence 全 CC-BY 的源，图片级过半是 **©版权保留 / CC-BY-NC-SA**：
- Norwegian（最大源）：媒体级 **55% "All rights reserved" + 17% CC-BY-NC-SA**，只 46% 真 CC-BY；
- naturgucker：媒体级 "Copyright by the creator, 见 URL"（GBIF 未解析出 CC-BY）；
- 相机陷阱源：媒体 license 空。

→ 初版 3.7M 清单 **54% 许可污染**，**不商用干净**。**修**：按 `multimedia.txt` 的**媒体级** license
严筛，只留真 CC0/CC-BY（©/NC/SA/未知全剔）→ **502 种 / 170 万图，license 100% CC0/CC-BY**。

> 教训：GBIF media 逐图 license 复核是**一等步骤**（01 §5 早写、03 记坑），本次实测证其严重性。

## 六、域检查（②合现实）

- **含相机陷阱图**（MACHINE_OBSERVATION，RECONYX）——域完美贴喂食器（鸟小、自然背景、动检触发）；
- **反面**：相机陷阱有**空帧/误触发**（动检触发但鸟已离开/太糊）→ 裁框步 `--drop-no-bird` 丢弃（不 center-crop 毒标）；
- 分组抽查（花园鸟/猛禽/涉禽/尾种）种标全对；
- 仍有域 gap（网络好图 vs 现场糊图），靠**自采 feeder** 补（未启动，08）。
