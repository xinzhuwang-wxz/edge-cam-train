**组装**（`python -m edge_cam.data.adapters.classify.build --config configs/data/classify_feeder.yaml`，CPU）：

## 2026-06-22 · 整图 V1 → 检测裁框 V2（关键提分）+ envelope + 地域/月份

> 全过程在 5090（克隆卡 westc）；完整图文报告 `results/classify/分类实验报告.{md,html}`（无实验1，避免干净集高指标误解）。

**V1（整图）踩坑 → 修 harness**：首训 balanced_sampler 在 ep10-13 触顶 val 0.56 即过拟合（train 0.88，val_loss 回升）；
且发现训练**无 best-on-val checkpoint**（Lightning 默认只存末轮）、`export_classifier` 导出内存末轮模型 → 过拟合时导出最差轮。
→ 改 `train.py`：加 `ModelCheckpoint(monitor=val_top1)` + `EarlyStopping` + fit 后加载 best 再返回（export/评估用最优轮）；
`balanced_sampler→class_weighted`（反频 CE，不重复采样，减稀有类过拟合）+ wd 1e-3。整图 V1 best **val_top1 0.596**。

**V2（检测裁框，最大杠杆）**：抽样发现原图鸟只占画面 **3~38%**（robin 仅 3%、多鸟+水面常见）→ 整图缩 224 训练被背景稀释。
用 **feeder_416 检测器**裁每张训练图（最高分 bird 框 + `expand_to_square`+15% → 256² crop），**99.997% 命中**（crop_dataset.py，
ORT 无 GPU provider → 48 进程并行，~5min/11.5万）。重训（配方同 V1，仅换裁框输入）：

| | V1 整图 | **V2 裁框** |
|---|---|---|
| fp32 val top-1 | 0.596 | **0.748（+15.2pt）** |
| **int8 top-1（端侧）** | 0.567 | **0.750（量化≈零掉点）** |
| field 退化 | 0.370 | **0.626（+25.6pt）** |

→ 一举两得：鸟填满输入（消背景稀释）+ **训练=级联推理**（消 domain gap）。端侧 75.0%/86.7% 可部署。

**INT8 envelope**（`run_envelope`，ORT-QDQ）：fp32 0.748 / int8 0.750 / field 0.626；Lite0 对 INT8 友好（去 SE/swish）。

**级联联合推理**（`cascade_demo.py`：全图→检测裁+15%→Lite0，置信门控/回退）：20 种鸟检测 20/20、提交精度 **85%**，
逐步标注图见 `results/classify/cascade_v2/`。

**地域 / 月份 mask**（推理后处理，**不重训**；类标是学名 → 区域清单直接对学名，eBird 非必需）：
- eBird crosswalk `ebird_map.csv`（360 命中 347/96.4%，`scripts/build_ebird_mapping` 思路，公开 taxonomy）。
- 区域清单走 **GBIF occurrence facet（免 key，1 次/国）**——`scientificName` 参数不过滤（坑），改 `species/match→usageKey→facet=speciesKey`；
  `network_turbo` 拖慢非 github 流量 → GBIF 直连反而快（坑）。
- 正确口径 `evaluate_regional`（只在 in-region 子集比 mask on/off）：地域 GB+0.3 / DK+0.5 / US+1.6 / **AU+5.4pt**（越窄越大）；
  月份（DK 物候）1月+2.23 / 其余~+1.5pt，**在地域之上叠加**。脚本 `build_regions.py` / `build_months.py` / `regional_eval.py`。

**其他坑**：① onnx 导出/对齐前须 `model.cpu()`（load_from_checkpoint 后在 GPU、dummy 在 CPU → 设备不匹配崩）。
② Hydra override 值含 `=`（ckpt 文件名 `epoch=38`）会被解析炸 → 复制成无 `=` 名再传。
③ 非鸟动物级联示例未做（检测集 3090 已删）。
